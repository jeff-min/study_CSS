Box Type-Block

        Box Type : block / inline / inline-block  / flex 의 종류가 있다.

        block  == >  길을 막다의 개념을 알자!

        block 태그는 차곡차곡 쌓는 이미지인데 본인 다음으로 오는 block 을 옆자리에 오지 못하게 함
            



        Block 특징
        1. 따로 width를 선언하지 않은 경우 , width = 부모의 content-box의 100%

        2. 따로 width를 선언한 경우, 남은 공간은 margin으로 자동으로 채움
            
        
            
            
            
            
        
        3.box 모델에 있는 속성들은 다 적용이 가능하다 / width / height / padding / border .....

        4.따로 부모의 height 을 선언하지 않을 경우 , 자식 요소의 height의 합 = 부모의 height
