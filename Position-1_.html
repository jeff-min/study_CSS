Position-1_


        Position 에는 5가지의 종류가 있다.
        1.static
        2.relative
        3.absolute
        4.fixed
        5.sticky

        

        position 을 공부할 때는 2가지를 반드시 알아야한다.

                Type                 - >                   기준점
    어떤종류의 포지션을 사용하는지      ->  내가 사용하는 포지션은 무엇의 기준으로 움직이는지.

    1. static 은 모든 요소의 기본 position 값 (일반적인 상태.)

    2.  relative 의 기준점은 바로 자기자신이 원래 있었던 위치..!
        
                
    3.  absolute 는 float 와 비슷하다.
            
            
            
        
            absolute 무엇을 기준으로 삼아서 위치시키나?
            ==> 자신이 삼고싶은 기준점을 새로 삼을 수 있다.

            
            

            
            

            

    4.  fixed 는 absolute와 동일한 현상이 일어난다.
    

            단 차이점은 fixed는 viewport 사이즈 기준으로 자신을 위치시킴.

            
            

            이제  위치를 이동시킬때 사용하는 CSS property 는 top/ bottom / left / right 인 속성인데 
            중요한 Tip은 (가로 방향중) left /right 중 하나  (세로 방향중) top / bottom 중 하나만 사용하는게
            
            

        5. Z-index
            (나머지 알아할 CSS- property)

            position 된 요소들의 수직 방향으로서의 레벨  /  위치를 알려준다.
