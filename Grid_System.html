        어떤 방식으로 작업하는지 / 디자이너의 의도를 파악해야힘

        디자이너분들은 일반적으로 Grid_System 을 이용해서 디자인한다.
        Grid_System 의 기본적인     container   |   column   |   gutter

        Grid 는 보통 12칸으로 나눔  // 디자이너가 기준을 정하기 나름이지만 범용적으로 12 칸을 많이 이용함
                                    //  2,3,4,6 으로 다 나뉘어져서 layout 을 짜기 용이함

        container   :   그리드 시스템이 적용이 되는 범위 // 즉 전체 디자인 범위

        column  :   1칸의 요소크기를 말함 (12 칸을 나눴을때에)
                    1칸 차지하는 요소 : col-1   / 4칸 차지하는 요소 : col-4 이런식으로 표현해서 알려주심

        gutter  :   간격에 대한 여백
