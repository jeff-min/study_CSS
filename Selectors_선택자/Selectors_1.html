selector 선언문
        selector{
            property : value;
        }

        중요한건
        selector   - 원하는 녀석을 고르는 방법을 알아보자.

        Type , Class & ID Selectors

        Type Selector : HTML 태그 샐렉터
        HTML 태그 자체를 직접적으로 지칭

        Class Selector : 태그의 class 를 주고 class로 선택자를 잡음
    
        ID Selectors :태그의 id 를 주고 선택자로 잡음
