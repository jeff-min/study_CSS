CSS 하다보면 요소의 속성값이 바뀌어야 할 때가 생긴다.
        애니메이션 처럼 스르륵 변화를 줄 수 있게 해줌.

        4개자 속성을 알아야함.
        property    |   duration    |   [timing-function]   |   [delay]
                                        [timing-function]   |   [delay]  는 생락 가능
        
        property    : 어떤 속성을 변화 시킬건가
        duration    : 변화 지속시간         | 단위 ms / s

        

        [timing-function]  : 변화의 속도 지정
        ease-in     |   ease-out    |   ease-in-out     |   cubic-bezier()
        ease-in : 천천히 바뀌다 나중에 빠르게
        ease-out : 처음엔 빨리 나중에 천천히
        ease-in-out : 짬뽕
        cubic-bezier()  : 본인이 변화의 가속도를 재현하고 싶을때 사용
        
        [delay]  : 시간지연
