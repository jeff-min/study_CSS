Box Type-Inline

        Inline  ==> 흐름의 개념을 알자

        block 의 같은 경우 자신의 옆에 못오게 계속 길막을 하는 반면에
        inline 요소는 정반대 특징이다.
        가능한 부피만큼 옆으로 계속 흐른다.
        그러다 width 한정적인 크기에 직면하면아랫줄에서 다시 흐른다.

        
        
        
        
        Inline 특징
        1. 박스 모델 property 중에 width / height / padding-top / padding-bottom / 
                border-bottom / border-top / margin-bottom / margin -top  사용불가
